# 🚀 Gyana Social – Backend

A full-stack social media backend built using Spring Boot, designed to support authentication, posts, comments, reels, chat, and real-time communication.

---

## 📌 Features

- JWT Authentication & Authorization  
- User Management (Register, Login, Profile)  
- Posts (Create, Like, Comment)  
- Real-time Chat (WebSocket)  
- Reels & Stories  
- REST APIs for frontend integration  
- Exception Handling & API Response Standardization  

---

## 🛠️ Tech Stack

- Backend: Spring Boot  
- Database: MySQL  
- ORM: Hibernate / JPA  
- Security: Spring Security + JWT  
- Real-time: WebSocket  
- Build Tool: Maven
